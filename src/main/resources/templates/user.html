Smart Contact Manager

Smart Contact Manager is a web-based application designed to help users manage their contacts efficiently. With Smart Contact Manager, users can store their contacts securely on the cloud and access them from anywhere, anytime.

User Dashboard

Key features:

  • View contacts
  • Add Contact
  • See specific contact after clicking on any contact email
  • View own profile

Smart Contact Manager is developed using modern web technologies to provide a seamless user experience.
